# Visual HTML Designer

A premium, interactive web-based HTML designer with real-time preview and visual element manipulation.

![Visual Designer Interface](https://raw.githubusercontent.com/username/visual-html-editor/main/assets/preview.png) (Placeholder for screenshot)

## Features

- **Live Preview**: Real-time synchronization between the code editor and preview.
- **Visual Editing**:
    - **Selection**: Click elements in the preview to select.
    - **Draggable**: `Alt + Drag` to move elements.
    - **Resizable**: `Alt + Shift + Drag` to resize elements.
    - **Content Editing**: Edit text directly in the preview.
- **Asset Upload**: One-click local image upload or drag-and-drop onto images.
- **Premium Aesthetics**: Sleek dark mode with glassmorphism effects.

## Quick Start

1. Clone the repository.
2. Open `index.html` in any modern web browser.

## Technologies Used

- [Monaco Editor](https://microsoft.github.io/monaco-editor/) - The power behind VS Code's editor.
- [Lucide Icons](https://lucide.dev/) - Beautifully simple pixel-perfect icons.
- Vanilla JavaScript, HTML5, and CSS3.
